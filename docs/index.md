# Vue Storefront 2 integration with __replace_me__

This project is a __replace_me__ integration for Vue Storefront 2.

This integration is currently a **Beta** and not ready for production usage. You can help us to make the integration production-ready faster by reporting bugs and contributing to the code at the [repository issues page](https://github.com/vuestorefront/__replace_me__/issues).
